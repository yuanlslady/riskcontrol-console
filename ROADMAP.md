# Roadmap

> 目标：把“个人风控/纪律”做成可复用的流程产品，而不是一堆散落的表单。

## 近期（稳定性与可用性）

- 登录 / 注册 / 找回密码链路稳定性
- Demo/真实站的分流逻辑更清晰（入口与提示文案）
- OCR 导入失败时的错误提示与可恢复策略（避免卡死在 `Invalid JWT` / 超时）
- 性能与交互：大列表、编辑体验、加载状态、空状态

## 中期（工作流增强）

- 交易审查（Pre-trade）模板化与可配置（不同交易类型不同清单）
- 投后复盘结构化字段（可统计、可追踪）
- 事件与复盘的关联（Position / Watchlist / Thesis）
- 自动化（portfolio automation）更可控：触发条件、节奏、运行日志可视化

## 长期（影响力与协作）

- 可导出的“复盘报告”（Markdown/PDF）
- 公共 Demo 数据集与案例库（更容易分享、讨论、教学）
- 贡献者指南完善：需求看板、任务分级、规范化 PR 流程
